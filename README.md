暴雨洒向寂寞汉子

人缓步瑟缩冷风里面

内心中的苦楚刺痛更使我倦

梦已失去没甚意思

从前或不懂爱的意义

这个深夜里 难明白什么是情

错对或是缘

在那天说浪漫句子

曾甜蜜依恋往昔片段

让娇小的身躯靠向我亲我面

但这天你弄着雨衣

垂头未懂得说一个字

心爱的伴侣 无言含泪转身去
